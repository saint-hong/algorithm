- 여러가지 알고리즘 지식과 사례들을 정리하고
- 다양하고 좋은 코드들을 클론 코딩하고 분석하여 원리와 전개방식을 파악하고
- 내가 직접 원하는 조건과 상황에 맞는 효율적인 코딩을 하기 위한 목적
